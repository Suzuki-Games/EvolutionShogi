# Evolution Shogi

Unity (C#) で開発したターン制ストラテジーゲーム。

将棋をベースに「駒の成長（進化）」の要素を加えたオリジナルゲームです。プレイヤーの主人公駒は「歩」として始まり、敵を倒し前進することで経験値を貯め、**歩 → 銀 → 飛車 → 勇者（全方向走り）** へと進化します。

## Features

- **進化システム**: 経験値ベースの4段階進化。進化するたびに移動パターンが拡張される
- **非対称バランス**: 弱い自陣 vs 強力な敵陣。成長で逆転する展開
- **持ち駒（Mochigoma）**: 取った敵駒を再利用可能。二歩チェック付き
- **スコアベースAI**: 攻撃価値・リスク評価・位置評価で最善手を選ぶ敵AI
- **リスポーン**: 勇者が倒されても3ターン後に歩として復活

## Tech Stack

- Unity 6.3 / C#
- 14 scripts / 7x7 board

## 開発スタイル

AIツール（Claude Code）を活用したペアプログラミングで開発しました。
- **自分の役割**: コンセプト設計、非対称バランスの方針決定、プレイテストによる品質検証、バグ発見と報告、Unityエディタ上のシーン構築・プレハブ設定
- **AIの役割**: C#コード実装、バグ原因特定と修正、アーキテクチャ提案
- **学び**: AIを使いこなすにも「何を作りたいか」「何がおかしいか」を正確に伝える力が必要だと実感しました
